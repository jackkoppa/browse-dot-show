# listen, fair play

Searchable archive of Football Cliches podcast episodes

## Local Dev



```shell
# pre-reqs: Node, pnpm

git clone git@github.com:jackkoppa/listen-fair-play.git
cd listen-fair-play
pnpm i
pnpm run dev

# view @ http://localhost:5173
```
